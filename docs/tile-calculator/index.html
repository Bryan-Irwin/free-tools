Tile Calculator Tips

  • Always add 10% waste for straight layouts, 15% for diagonal or herringbone patterns
  • Buy all tiles from the same lot to ensure consistent color
  • Standard grout width is 1/8" for rectified tiles, 3/16" for standard tiles
  • Common tile sizes: 12x12", 12x24", 6x6", 4x4", 3x6" (subway)
  • Order extra tiles and keep them for future repairs
