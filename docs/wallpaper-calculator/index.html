Wallpaper Tips

  • Always buy an extra roll for pattern matching waste and future repairs
  • Patterned wallpaper with a large repeat requires more rolls — add 15-20% instead of 10%
  • Standard door opening: ~21 sq ft (3ft x 7ft)
  • Standard window opening: ~12 sq ft (3ft x 4ft)
  • Remove old wallpaper completely before applying new — do not layer
  • Peel-and-stick wallpaper is easier for beginners and renters
